## About Me

I'm a Senior Mobile Engineer with **7+ years** of end-to-end ownership across security-sensitive mobile products. I specialise in building apps where the stakes are high — regulated fintech, physical access control, and institutional education platforms — using **React Native**, **Flutter**, and **Android (Java/Kotlin)**.

My work sits at the intersection of engineering rigour and product clarity. I care deeply about:

- **Security-first architecture** — biometric auth, SSL pinning, OAuth 2.0, JWT, secure storage
- **Payment reliability** — PCI DSS flows, KYC/AML integrations, real-time transaction systems
- **CI/CD and release quality** — GitHub Actions, Codemagic, Fastlane, 98%+ test pass rates
- **Clean, maintainable codebases** — Clean Architecture, MVVM, feature-based modularisation

*Continuous improvement and a beginner's mindset keep me shipping better software.*
